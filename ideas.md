## المقاربة المختارة: الإثارة الليلية (Dark Tournament)

### Design Movement
Dark Sports Dashboard — مستوحى من لوحات تحكم الرياضة الاحترافية والألعاب التنافسية

### Core Principles
1. **الكحلي العميق كأساس** — يمنح الموقع هيبة وعمقاً بصرياً
2. **البرتقالي كلون الانتصار** — يُستخدم للتمييز والتحفيز والعناصر الحيوية
3. **البيانات أولاً** — كل قسم يعرض معلومات واضحة وقابلة للقراءة
4. **الحركة الهادفة** — انيميشن خفيف يعكس الديناميكية التنافسية

### Color Philosophy
- **الخلفية الرئيسية:** `#0D1B2A` (كحلي ليلي عميق)
- **الكحلي الثانوي:** `#1B2E45` (بطاقات وعناصر)
- **البرتقالي الأساسي:** `#F7941D` (لون يراع وتواق 2)
- **البرتقالي الفاتح:** `#FFB347` (hover وتدرجات)
- **الأبيض/الرمادي الفاتح:** `#E8EDF2` (النصوص الرئيسية)
- **الرمادي المتوسط:** `#8FA3B8` (النصوص الثانوية)

### Layout Paradigm
- Sidebar ثابت على اليمين (RTL) للتنقل
- المحتوى الرئيسي يأخذ باقي المساحة
- بطاقات متدرجة الأحجام لإبراز الترتيب
- لوحة بيانات حية مع إحصائيات

### Signature Elements
1. **بطاقة الترتيب الذهبية** — المركز الأول بإطار برتقالي متوهج
2. **شارة الاسم المستعار** — أيقونة مميزة لكل لاعب
3. **شريط الأحداث المتحرك** — يعرض آخر الأحداث في الوقت الفعلي

### Interaction Philosophy
- Hover effects سلسة على البطاقات
- أزرار تستجيب بنبضة خفيفة عند الضغط
- انتقالات سريعة بين الصفحات

### Animation
- Entrance: fade-in + slide-up (200ms ease-out)
- Cards: scale(1.02) on hover (150ms)
- Buttons: scale(0.97) on active (100ms)
- Leaderboard: stagger 50ms per row

### Typography System
- **العناوين:** Noto Kufi Arabic (Bold) — قوي وعربي أصيل
- **النصوص:** Noto Sans Arabic (Regular/Medium) — واضح ومقروء
- **الأرقام:** Tajawal (Bold) — أرقام عربية جميلة

### Brand Essence
**تواق 2** — منصة التنافس العلمي للجيل الطموح. للطلاب الذين يسعون للتميز. الفرق: الإنجاز العلمي يُحوَّل إلى مغامرة تنافسية.

### Brand Voice
- **الشعار:** "جيل يعلو ... وبالعلم يسمو"
- **CTA:** "ابدأ تحديك الآن"
- **Microcopy:** "أنت الصقر، أنت البرق، أنت الذئب"

### Signature Brand Color
`#F7941D` — البرتقالي الذهبي لتواق 2
